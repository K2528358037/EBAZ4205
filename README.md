#### 硬件资源

1.  主控：    ZYNQ XC7Z010-1CLG400I
2.  DDR：     128M x 16bit
3.  Flash：   128M的NAND FLASH
4.  网络：    1个百兆网口PL端
5.  串口：    2个串口（PS端TTL+PL端USB转232）
6.  蜂鸣器：  无源蜂鸣器1个
7.  LCD：    240*240分辨率 SPI接口IC：ST7789v
8.  LED：    5个（PS端2个，PL端3个）
9.  HDMI：   1路HDMI（PL端）
10. 按键：   7个按键（两个PS端1个未焊接，5个PL端）
11. SD卡座： 1个
12. 启动模式：SD卡启动
